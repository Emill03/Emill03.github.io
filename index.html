Elementos HTML
html: Define el documento HTML.
head: Contiene metadatos y enlaces a scripts y hojas de estilo.
body: Contiene el contenido visible de la página web.
h1 a h6: Representan encabezados, donde h1 es el más importante y h6 el menos.
p: Define un párrafo de texto.
a: Define un enlace hipervínculo.
ol y ul: ol crea una lista ordenada y ul una lista desordenada.
strong: Define texto con énfasis fuerte, generalmente se muestra en negrita.
em: Define texto enfatizado, generalmente se muestra en cursiva.
form: Define un formulario para la entrada de usuario.
hr: Inserta una línea horizontal para separar contenido.
input: Define un control interactivo en un formulario, como un campo de texto.
footer: Define el pie de página de un documento o sección.
br: Inserta un salto de línea.
Elementos CSS
color: Establece el color del texto.
font-family: Define la familia de fuentes del texto.
font-size: Establece el tamaño de la fuente.
font-weight: Define el grosor de la fuente (e.g., normal, bold).
text-align: Establece la alineación del texto (e.g., left, center).
background-color: Define el color de fondo de un elemento.
width: Establece el ancho de un elemento.
height: Establece la altura de un elemento.
